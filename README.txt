Instructions
============

1. Install SIMBL_
2. Install the `TerminalColours SIMBL Plugin`_
4. Double-click on `Strange.terminal` to load the new color scheme
5. Open the Terminal.app settings and make `Strange` the default profile

.. _SIMBL:
   http://www.culater.net/software/SIMBL/SIMBL.php

.. _TerminalColours SIMBL Plugin:
    http://github.com/timmfin/terminalcolours/raw/master/TerminalColours-SL-64bit.zip
