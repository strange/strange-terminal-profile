A Terminal.app theme based on `IR_Black` by Todd Werth
(http://blog.infinitered.com/entries/show/6).

Instructions
============

1. Install SIMBL_
2. Install the `TerminalColours SIMBL Plugin`_
3. Tick the "Open in 32-bit mode" checkbox in the Terminal.app info dialog
4. Double-click on `Strange.terminal` to load the new color scheme
5. Open the Terminal.app settings and make `Strange` the default profile

.. _SIMBL:
   http://www.culater.net/software/SIMBL/SIMBL.php

.. _TerminalColours SIMBL Plugin:
   http://cloud.github.com/downloads/evanphx/terminalcolours/TerminalColours-SL.tar.gz
